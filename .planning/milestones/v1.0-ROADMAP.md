## Overview

Claude Notify 是一个 Claude Code 全局技能,通过 Pushover 和 Windows 系统通知为所有项目提供任务完成通知功能。让开发者在 Claude Code 执行任务时能够离开电脑,在任务完成或需要关注时及时收到通知。

**Key Decisions:**

1. **Async Hook Execution** - Set async: true to prevent blocking Claude Code responses
2. **Parallel Notification Architecture** - ThreadPoolExecutor for simultaneous notifications
3. **Timeout Cascade Strategy** - Strict timeouts at each layer (Claude 2s, Pushover 2s, Windows 1s, overall 4s)
4. **Graceful Degradation Pattern** - Failures fall back to fixed message template
5. **Fast Failure Strategy** - No retry logic to ensure 5-second completion guarantee
6. **Marketplace Integration** - Add plugin to existing work-skills marketplace
7. **Plugin-Level Hook Format** - Use description + hooks wrapper format
8. **Path Portability Pattern** - Use ${CLAUDE_PLUGIN_ROOT} for all internal file references
9. **Environment Variable Pattern** - os.environ.get() with None check and warning
10. **Feature Flag Pattern** - Path.is_file() check for notification control
11. **Automatic Cleanup Pattern** - Run log cleanup at startup with 5-day retention
12. **Diagnostic Verification** - Send actual test notification for end-to-end connectivity check
13. **Credential Masking** - Show first 4 and last 4 chars only for security
