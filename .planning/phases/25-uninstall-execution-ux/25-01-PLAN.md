    **Core function `removeAllComponents(detectionResults)`:**
    Returns `Promise<Array<{category: string, name: string, status: 'removed'|'failed'|'skipped', detail: string}>>`

    Execute 7 steps in order per D-07. Each step uses a `removeStep(category, name, fn, shouldRun)` helper that:
    - If `shouldRun` is false: returns `{ category, name, status: 'skipped', detail: '' }`
    - If `shouldRun` is true: calls `fn()` inside try/catch. Success = `status: 'removed'`, failure = `status: 'failed'` with `detail: error.message`
    - NEVER throws -- all errors are caught and recorded (D-08, UX-05)

    **Step 1 - Hook Scripts:** Check `detectionResults.hooksScripts.installed`. If true, delete `notify-stop.py` and `notify-attention.py` from `getHooksDir()`. Use `fs.rmSync(p, { force: true })` for each file. Name: `'Hook scripts'`.

    **Step 2 - Hook Registration:** Check `detectionResults.hooksRegistered.installed`. If true, call `_readSettings()`, pass result through `_removeExistingNotifyHooks()`, then `_writeSettings()`. Name: `'Hook registration'`.

    **Step 3 - Slash Commands:** Check `detectionResults.commandsInstalled.installed`. If true, delete these 4 files from `getCommandsDir()`: `notify-enable.md`, `notify-disable.md`, `notify-status.md`, `check-notify-env.md`. Use `fs.rmSync(p, { force: true })` for each. Name: `'Slash commands'`.

    **Step 4 - Plugin Directories:** For each plugin in `detectionResults.plugins`, if `plugin.installed` is true, delete `path.join(getSkillsDir(), plugin.name)` using `fs.rmSync(pluginPath, { recursive: true, force: true })`. Each plugin gets its own result entry with name `plugin.name`.

    **Step 5 - Marketplace Cache:** Check `detectionResults.marketplaceSource.installed`. If true, delete `path.join(os.homedir(), '.claude', 'plugins', 'cache', 'work-skills')` and `path.join(os.homedir(), '.claude', 'plugins', 'marketplaces', 'work-skills')` using `fs.rmSync(dirPath, { recursive: true, force: true })` for each. Use `fs.existsSync` check before each deletion. Name: `'Marketplace cache'`.

    **Step 6 - Marketplace Source:** Check `detectionResults.marketplaceSource.installed`. If true, call `readClaudeConfig()`, delete `config.marketplaceSources['work-skills']` if it exists, then `writeClaudeConfig(config)`. Name: `'Marketplace source'`.

    **Step 7 - Environment Variables:** For each key in `['token', 'user']` from `detectionResults.envVars`, if `envVar.installed` is true, execute `await execa('reg', ['delete', 'HKCU\\Environment', '/v', envVar.name, '/f'])`. Each env var gets its own result entry with name `envVar.name`. Use `/f` flag to skip Windows confirmation prompt (per RESEARCH.md Pitfall 3 guidance).
