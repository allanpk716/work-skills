<objective>
Flatten windows-git-commit plugin directory structure so SKILL.md is at plugin root level, matching claude-notify's working structure.

Purpose: The installer's isPluginInstalled() checks for SKILL.md at ~/.claude/skills/<name>/SKILL.md. Currently windows-git-commit has SKILL.md nested two levels deep (skills/windows-git-commit/SKILL.md), so after cpSync the installer cannot detect the installed plugin. Flattening to match claude-notify's structure fixes this without modifying installer code.

Output: Restructured plugin directory with SKILL.md, hooks/, scanner/, tests/ all at plugin root level.
</objective>
