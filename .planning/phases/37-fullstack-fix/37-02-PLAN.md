Replace with the proven implementation from pyts-calculator that writes meta:

```python
def receive(entry: dict[str, Any]) -> bool:
    """Write a frontend code point entry. Thread-safe. Returns False if disabled."""
    if not _ts_enabled or _ts_out is None:
        return False
    name = entry.get("name", "unknown")
    stack = entry.get("stack", "")
    meta = entry.get("meta")
    with _ts_lock:
        if meta:
            import json as _json
            _ts_out.write(f"[CODEPOINT] {name}\n{stack}\nmeta: {_json.dumps(meta)}\n")
        else:
            _ts_out.write(f"[CODEPOINT] {name}\n{stack}\n")
        _ts_out.flush()
    return True
```

The key change is: extract `meta` from the entry dict, and if it exists, write it as a JSON line after the stack. This ensures flow_id (inside meta) is available for cross-language correlation.
