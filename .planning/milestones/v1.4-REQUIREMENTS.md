# Requirements: Work Skills v1.4

**Defined:** 2026-03-29
**Core Value:** 为 Windows 开发者提供即开即用的 Claude Code 技能,自动化重复性工作,让开发者专注于创造性任务

## v1 Requirements

### 插件检测修复

- [x] **DETECT-01**: windows-git-commit 安装后 `isPluginInstalled()` 能正确检测到 SKILL.md 存在
- [x] **DETECT-02**: 重复运行安装器时,windows-git-commit 显示 `[installed]` 标记且不提示安装
- [x] **DETECT-03**: 修复后 claude-notify 的安装检测仍正常工作(回归验证)

### 插件结构一致性

- [x] **STRUCT-01**: windows-git-commit 插件结构与安装器检测逻辑保持一致
- [x] **STRUCT-02**: 所有插件的 SKILL.md 安装后都在 `~/.claude/skills/<name>/SKILL.md` 路径可访问
