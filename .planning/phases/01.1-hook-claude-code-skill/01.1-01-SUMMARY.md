### Pushover Configuration (Optional - required for mobile notifications)

**Environment variables to add:**
- `PUSHOVER_TOKEN`: API token from Pushover Dashboard → Your Applications → Create Application
- `PUSHOVER_USER`: User key from Pushover Dashboard

**How to configure (Windows):**
```cmd
setx PUSHOVER_TOKEN "your-pushover-app-token"
setx PUSHOVER_USER "your-pushover-user-key"
```

**Verification command:**
```bash
python scripts/verify-installation.py
```

**Note:** Windows Toast notifications work without any configuration. Pushover is only needed for mobile push notifications.
