  <action>
    Create `scanner/rules/whitelist.py` with the following components:

    1. **WhitelistDirective dataclass**
       ```python
       @dataclass
       class WhitelistDirective:
           directive_type: str  # ignore-line, ignore-file, ignore-rule, ignore-category
           rule_id: Optional[str] = None
           line_number: Optional[int] = None
       ```

    2. **parse_whitelist_comments() function**
       - Input: file content (string)
       - Output: List[WhitelistDirective]
       - Algorithm:
         a. Split content into lines
         b. For each line (starting at line 1):
            - Search for pattern: `#\s*gitcheck\s*:\s*(ignore-\w+(?::\w+)?)`
            - Case-insensitive, allow extra spaces
            c. Parse directive type:
            - "ignore-line" → WhitelistDirective('ignore-line', line_number=N)
            - "ignore-file" → WhitelistDirective('ignore-file')
            - "ignore-rule:INTL-01" → WhitelistDirective('ignore-rule', rule_id='INTL-01')
            - "ignore-all-ips" → WhitelistDirective('ignore-category', rule_id='INTL-01')
            - "ignore-all-emails" → WhitelistDirective('ignore-category', rule_id='INTL-03')
         d. Return list of all directives found

    3. **should_skip_detection() function**
       - Input: rule_id (str), line_number (int), directives (List[WhitelistDirective])
       - Output: bool (True = skip this detection)
       - Priority order (first match wins):
         1. ignore-file (anywhere in file)
         2. ignore-rule with matching rule_id
         3. ignore-line with matching line_number
         4. ignore-category with matching rule_id

       ```python
       def should_skip_detection(
           rule_id: str,
           line_number: int,
           directives: List[WhitelistDirective]
       ) -> bool:
           for directive in directives:
               if directive.directive_type == 'ignore-file':
                   return True
               if (directive.directive_type == 'ignore-rule' and
                   directive.rule_id == rule_id):
                   return True
               if (directive.directive_type == 'ignore-line' and
                   directive.line_number == line_number):
                   return True
               if (directive.directive_type == 'ignore-category' and
                   directive.rule_id == rule_id):
                   return True
           return False
       ```

    4. **Export symbols**
       - __all__ = ['WhitelistDirective', 'parse_whitelist_comments', 'should_skip_detection']

    **Key implementation notes:**
    - Use loose regex matching (case-insensitive, allow spaces)
    - Line numbers start at 1 (not 0)
    - Multiple directives in one file are allowed
    - Follow RESEARCH.md Pattern 2 exactly
