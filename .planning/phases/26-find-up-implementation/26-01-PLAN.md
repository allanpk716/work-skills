    The module structure:
    ```python
    import logging
    from pathlib import Path
    from typing import Dict, Optional

    logger = logging.getLogger(__name__)

    def check_notification_flags() -> Dict:
        """
        Check for notification disable flags, searching from CWD upward.

        Traversal rules (per D-01 through D-05):
        1. Start at CWD, check .no-pushover and .no-windows at each level
        2. Each channel tracked independently (D-02 strict reading)
        3. If CLAUDE.md found at a level where NEITHER .no-xxx was found, stop entirely (D-03)
        4. If CLAUDE.md found but one .no-xxx WAS found, continue searching for the other channel
        5. Maximum 10 levels (D-04)
        6. Stop at filesystem root (parent == self)

        Returns:
            dict: {
                'pushover_disabled': bool,
                'windows_disabled': bool,
                'pushover_path': Optional[Path],
                'windows_path': Optional[Path]
            }
        """
    ```

    Implementation MUST use a while loop (not recursion, not Path.parents iteration) per the RESEARCH.md recommendation.

    Key algorithm details:
    - `current = Path.cwd()`, `depth = 0`, `max_depth = 10`
    - At each level, check `(current / '.no-pushover').is_file()` and `(current / '.no-windows').is_file()` independently
    - After checking flags, check if BOTH channels already found -> break early
    - Then check CLAUDE.md: `(current / 'CLAUDE.md').is_file()`
    - CRITICAL for D-02/D-03 interaction: Only stop on CLAUDE.md if at this level NO new .no-xxx was found. Implementation approach:
      - Track `found_pushover_this_level` and `found_windows_this_level` booleans
      - If CLAUDE.md exists AND neither was found this level AND both channels not yet found -> break
      - If one channel was found this level but CLAUDE.md also exists, the other channel's search continues upward
    - Move up: `parent = current.parent`, if `parent == current` -> break (filesystem root)
    - `current = parent`, `depth += 1`
    - If `depth > max_depth` -> break

    Log results:
    - If pushover found: `logger.info(f"Pushover disabled by {pushover_path}")`
    - If windows found: `logger.info(f"Windows disabled by {windows_path}")`

    Return the dict with all 4 keys.

    After creating flags.py, run all test_flags.py tests to confirm GREEN.
